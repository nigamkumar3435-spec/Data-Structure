## 📖 Overview

This repository documents my journey of learning **Java Programming** and **Data Structures**. It is organized lecture-wise and includes implementations of core Java concepts, Object-Oriented Programming (OOP), arrays, collections, and foundational data structure concepts through hands-on coding exercises.
