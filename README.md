## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/nigamkumar3435-spec/java.git
```

Navigate to the project:

```bash
cd java
```

Compile a Java program:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```
