# 📖 Overview

This repository documents my learning journey in **Java Programming**, **Object-Oriented Programming (OOP)**, and **Data Structures**. It is organized lecture-wise, with each lecture introducing a new concept through practical Java programs.

The repository begins with Java fundamentals and gradually progresses to arrays, collections, constructors, encapsulation, inheritance, abstraction, interfaces, and access modifiers, creating a strong foundation for Data Structures and Algorithms (DSA).

* Learn Java programming from scratch

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/nigamkumar3435-spec/Data-Structure.git
```

Navigate to the project

```bash
cd Data-Structure
```

Compile a Java file

```bash
javac FileName.java
```

Run the program

```bash
java FileName
```
